在 Mac OS X 上安裝與使用 FIGlet

Jan 2, 2016 Mac English

FIGlet 是一個很有趣的軟體，它可以讓你利用 ASCII 字元建立好看的 Text Banner ( 若想進一步了解，請搜尋 ASCII Art )。當然 FIGlet 在其他 UNIX-like 系統上也可以安裝、使用，但本文章主要記錄如何在 Mac OS X 上安裝、使用 FIGlet。



安裝 FIGlet

基本上，透過 UNIX-like 系統的套件管理工具 ( Package Manager ) 都可以找得到 FIGlet，若在 Mac OS X 上，我們可以使用 Homebrew 來安裝。指令如下：

# 搜尋 FIGlet
brew search figlet

# 取得 FIGlet 相關資訊
brew info figlet

# 安裝 FIGlet
brew install figlet


使用 FIGlet

使用 FIGlet 方式很簡單，以下列出幾個常用的指令：

# 輸出訊息 ( 如：Devforgalaxy )
figlet Devforgalaxy

                                |___/                    |___/

# 輸出訊息並置中
figlet -c Devforgalaxy

# 輸出字元保有間距
figlet -k Devforgalaxy

# 輸出斜體訊息
figlet -f slant Devforgalaxy

# 輸出 Shadow 字體訊息
figlet -f shadow Devforgalaxy

更多的字體可以參考這裡，字體檔名為 *.flf，其存放路徑是 Homebrew 安裝的預設目錄 ( 如：/usr/local/Cellar/figlet/2.x.x/share/figlet/fonts )。
