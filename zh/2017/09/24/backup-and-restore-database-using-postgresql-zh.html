使用 PostgreSQL 備份與復原資料庫

Sep 24, 2017 PostgreSQL English

在前一篇文章中，我們了解到如何使用 PostgreSQL 建立資料與使用者。 除此之外，如何備份與復原資料庫也是一個開發者想要知道的，對吧？ 因此，本篇將介紹如何備份與復原資料庫。



備份資料庫

備份資料庫有許多方式，這邊將要介紹最簡單的方式：將資料庫輸出成一個文字檔案。指令如下：

pg_dump [資料庫名稱] > /path/to/[備份檔案名稱].txt


復原資料庫

要復原資料庫，你必須要先建立一個空的資料庫：

psql -c "CREATE DATABASE [資料庫名稱];"

建立好資料庫後，接著將備份檔的資料寫回新的資料庫：

psql [資料庫名稱] < [備份檔案名稱].txt

基本的備份與復原的方式就是這樣，別忘了定期備份你的資料。
