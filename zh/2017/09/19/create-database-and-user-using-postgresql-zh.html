使用 PostgreSQL 建立資料庫與使用者

Sep 19, 2017 PostgreSQL English

從 2016 年起 PostgreSQL 逐漸受到開發者青睞，許多開發者也將它視為資料庫首選。其受歡迎的原因不僅僅是它方便使用，它更提供許多有用的程式與指令。因此，本篇文章將介紹如何透過 PostgreSQL 提供的程式與指令輕鬆地建立資料庫與使用者。



建立使用者

要使用 PostgreSQL 提供的工具，你必須將目前使用者切換成 postgres。切換後，新增使用者：

createuser [使用者名稱]

若你有安裝 sudo 則可以：

sudo -u postgres [使用者名稱]

接著，我們需要設定該使用者密碼：

psql -c "alter user [使用者名稱] with encrypted password '[密碼]';"


建立資料庫

一樣地使用 postgres 使用者身份繼續建立資料庫：

createdb [資料庫名稱]

同樣地，若有安裝 sudo 則可以：

sudo -u postgres createdb [資料庫名稱]

建立好資料庫後，給予使用者權限來操作資料庫：

psql -c "grant all privileges on database [資料庫名稱] to [使用者名稱];"

現在，資料庫已經就緒，你可以開始使用它來開發。
