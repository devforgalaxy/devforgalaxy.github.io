替 PyPy 安裝套件管理工具：pip

Mar 13, 2017 Python Read in English

隨著 PyPy 社群與整個生態日趨成熟，未來將會有更多機會在新的專案中使用它，所以將如何替 PyPy 安裝 pip (套件管理工具) 與其使用方式做個簡單的紀錄。


前置準備

在開始安裝 pip 前，先確認你已經將 PyPy 安裝於欲使用的環境或機器上。接下來的步驟將在 Debian 上示範，安裝 PyPy 指令為：

# 更新 Debian 套件工具
apt-get update

# 安裝 PyPy 與 curl
apt-get -y install curl pypy


安裝 pip

安裝完 PyPy 後，接著來安裝 pip：

# 安裝 pip
curl https://bootstrap.pypa.io/get-pip.py | pypy

# 更新 pip
pypy -m pip install --upgrade pip


使用 pip

以下為 pip 的安裝與移除模組使用方式：

# 安裝第三方模組
# 範例: pypy -m pip install [module-name]
pypy -m pip install requests

# 移除模組
# 範例: pypy -m pip uninstall [module-name]
pypy -m pip uninstall requests
