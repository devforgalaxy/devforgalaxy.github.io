在 macOS 中透過 Command Line 安裝 .dmg 檔案

Oct 6, 2017 Mac English

MacOS 已經將安裝軟體的過程簡單化。然而，在某些情況下我們還是必須要透過 Command Line 安裝 .dmg 檔案。因此，本篇文章將介紹如何在 4 個步驟內完成安裝。



透過 Command Line 安裝 .dmg 檔案

整個安裝的過程一點也不複雜，基本上來說我們只需要將 .dmg 檔案掛載，接著把檔案全部複製到另一個資料夾，最後將 .dmg 卸載即可。暸解後，首先來掛載 .dmg 檔案：

sudo hdiutil attach [disk-image-name].dmg

進入掛載檔案：

cd /Volumes/[disk-image-name]

進入後，你可以看到 .app 資料夾，我們只要把該資料夾整個複製到 /Applications 就算安裝完成：

sudo cp -rf [app-name].app /Applications

最後把 .dmg 檔案卸載：

sudo hdiutil detach /Volumes/[disk-image-name]

現在，你應該可以開始使用剛剛安裝好的軟體。
