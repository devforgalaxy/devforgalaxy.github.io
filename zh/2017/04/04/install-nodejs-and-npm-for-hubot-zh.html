建立屬於自己的 Hubot (1) - 安裝 Node.js 與 npm

Apr 4, 2017 Hubot Read in English

Hubot 是一個由 Github 團隊所開發的自動化工具 (框架)，我們可以很輕鬆地利用它結合其他服務 (如：Google Map、Skype 與 Slack) 來提升我們的工作效率。接下來，我將會透過三篇文章來從頭介紹如何建立屬於你自己的 Hubot 並將它結合 Slack，第一篇將介紹如何設定 Hubot 所需環境。


前置準備

接下來的步驟將都會在 Debian 上示範，所以在開始安裝 Node.js 與 npm 之前，我們先更新套件管理工具與下載所需軟體 cUrl：

# 更新 Debian's 套件管理工具
apt-get update

# 安裝 cUrl
apt-get -y install curl


安裝 Node.js 與 npm

若你直接透過 apt-get install -y nodejs 安裝 Node.js，你會下載到較舊的版本。所以若要安裝較新的版本我們可以使用：

# 安裝 Node.js 6.X
curl -sL https://deb.nodesource.com/setup_6.x | bash -
apt-get -y install nodejs

# 安裝 Node.js 7.X
curl -sL https://deb.nodesource.com/setup_6.x | bash -
apt-get -y install nodejs


確認安裝版本

安裝好後，確認一下我們安裝的版本，指令如下：

# 查看 Node.js 版本
nodejs -v

# 查看 npm 版本
npm -v
