Install Docker on Amazon Linux

Feb 26, 2017 AWS 中文版

Amazon already introduced EC2 Container Service, but I guess many people might still need to install Docker on Amazon Linux manually. This post is about how to install Docker on Amazon Linux.



Prerequisite

Before we start it, make sure you create an EC2 instance using Amazon Linux AMI. If it’s up and running, let’s SSH to your instance:

# Set the pem file as read-only
chmod 400 /path/to/your/ec2.pem

# Default user is ec2-user
ssh -i "ec2.pem" ec2-user@ip


Install Docker

Let’s install Docker via yum:

# Update
sudo yum update -y

# Install Docker
sudo yum install -y docker

# Check Docker version
sudo docker -v

# Check Docker information
sudo docker info

To execute Docker commands without sudo, you need to add the user into Docker group. Like this:

sudo usermod -a -G docker ec2-user


Manage Docker daemon

Here are commands for managing your Docker daemon:

# Start Docker
sudo service docker start

# Stop Docker
sudo service docker stop
